Margin - Border - Padding

- The box model -

The box model is the friendly way to talk about design and layout in CSS.

We call it the box model since every element that we use in HTML is "wrapped" around a set of boxes, we use these boxes to manupulate how we desire this elemnt to look and behave.

As shown above the box layers for every HTML element are:

Margin: Transparent layer around the border clear space around the border.

Border: Outer layer of the element the limit of it before the margin.

Padding: Transparent layer to clear space around the element, I like to think of it as puff or fluff around your element, how squishy you like it.

Content: The insides of your box, where text and images are.

What HTML elements, Shrek and onions have in common?
They all have layes!!


Lets see how an element will interact with another one:

Lets imagine we have two buttons one next to another, these buttons have the same word in them and we will only modify the layers around them to see how they react.
the distance between them will be defined by the values of the addition of padding, border and margin.

Here the height of both buttons is the same since the addition of the values comes to the same number, but the distance between them differs because the margin of the right side button is bigger for both of the sides, having a distance between element of 28px.
