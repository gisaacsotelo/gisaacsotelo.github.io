- small definition of topic -

The box model is the friendly way to talk about design and layout in CSS.

We call it the box model since every element that we use in HTML is "wrapped" around a set of boxes, we use these boxes to manupulate how we desire this elemnt to look and behave.

As shown above the box layers for every HTML element are:

Margin: Transparent layer around the border clear space around the border.

Border: Outer layer of the element the limit of it before the margin.

Padding: Transparent layer to clear space around the element.
I like to think of it as puff or fluff around your element, how squishy you like it.

Content: The insides of your box, where text and images are.
